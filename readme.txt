This Plugin shows the user a jQuery climate table of the chosen city of crete. The plugin owner can choose between the temperatures celsius and fahrenheit. 

== Description ==

The user has to choose a month and a city and the plugin shows the maximum temperature, minimum temperature, water temperature, sunshine hours / day and rainy days / month.

#### Shortcode
* You can add Climate on Crete via shortcode [climate_on_crete]
* If you want to hide the backlink use the shortcode [climate_on_crete link="false"]

#### Widget
* Just insert the widget in the chosen sidebar, type a title and activate the wanted options

== Installation ==

1. Upload  "climate-on-crete" folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Climate on Crete to your widget enabled space
4. If you need insert in article please use short tag [climate_on_crete]
